# makiato
![alt tag](https://travis-ci.org/victorzki/makiato-fe.svg?branch=master "Travis status")

Makiato application.

### Installation
```
$ npm install
$ bower install
```

### Run the app
Start the express app with supervisor (to restart server on file changes)
```
$ supervisor --debug ./app/bin/www
```

Run gulp
```
$ gulp serve
```

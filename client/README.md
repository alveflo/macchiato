# makiato
[![Build status](https://travis-ci.org/victorzki/makiato.svg?branch=master)](https://travis-ci.org/victorzki/makiato)

Makiato front-end

### Installation
```
$ npm install
$ bower install
```

### Run
```
- Build
$ gulp build

- browserSync (+ fs watcher)
$ gulp serve

- Build + browserSync
$ gulp

```
